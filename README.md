# Deepfake Liveness Detector

Lightweight webcam-based liveness check that uses eye-blink detection with MediaPipe Face Mesh.

## Requirements

- Python 3.9+
- A working webcam

## Setup

1. Create and activate a virtual environment (optional).
2. Install dependencies:
   ```bash
   pip install opencv-python mediapipe
   ```

## Run

```bash
python deepfake_liveness_detector.py
```

Press `q` to quit the window.

## Notes

This project is a simple demo and is not intended to be a production-grade anti-spoofing system.
